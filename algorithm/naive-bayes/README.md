# 朴素贝叶斯

## 贝叶斯公式

<div align="center"><img src="http://latex.codecogs.com/svg.latex?P(A\cap&space;B)=P(A)*P(B|A)=P(B)*P(A|B)" /></a></div>

变形公式:

<div align="center"><img src="http://latex.codecogs.com/svg.latex?P(B|A)=\frac{P(A|B)*P(B)}{P(A)}" /></a></div>

* 先验概率:
    * P(A): A的先验概率
    * P(B): B的先验概率
* 后验概率:
    * P(A|B): B条件下A的后验概率
    * P(B|A): A条件下B的后验概率
